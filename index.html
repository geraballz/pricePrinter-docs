Características

Todo lo que necesita
tu imprenta

Diseñado específicamente para impresoras de tanque de tinta donde el costo real depende de cuánta tinta se consume, no de un precio fijo.

📄
Análisis página por página
Cada página se analiza individualmente. Los PDFs multi-página se procesan completos con un desglose detallado.
🎨
Detección real de color
Detecta píxeles cromáticos excluyendo negro/casi-negro. Evita falsos positivos en texto oscuro y fondos neutros.
📊
Precio lineal por cobertura
El precio escala proporcionalmente a la tinta real consumida. Una página con 13% de cobertura cuesta la mitad que una con 26%.
🔀
Precio mixto color
Las imágenes a color se cotizan según la proporción real de píxeles cromáticos. No se cobra el máximo a color si la imagen es mayormente gris.
☑️
Control por checkbox
Excluye páginas del total con un clic. Ideal para portadas, índices o páginas que el cliente ya tiene impresas.
💾
Tarifas persistentes
Las tarifas se guardan automáticamente en AppData. Sin necesidad de reconfigurar cada vez que abres la app.
🖥️
Windows y macOS
Compatible con Windows 10/11 y macOS Intel y Apple Silicon. Interfaz moderna con modo oscuro siempre activo.
📂
Drag & Drop
En Windows, arrastra archivos directamente sobre la lista para agregarlos. Soporta PDF, PNG, JPG, BMP, TIF y más.

Cómo funciona

Cuatro pasos
para cotizar

Sin complicaciones. Configura una vez y cotiza en segundos.

Configura tus tarifas
Define los precios por tipo de página y el punto de referencia para imágenes. Guárdalos una sola vez.
Carga los documentos
Arrastra los archivos o usa el botón Agregar. Puedes cargar múltiples PDFs e imágenes al mismo tiempo.
Analiza
PrintPricer examina cada página: detecta el tipo de contenido, mide la cobertura de tinta y calcula el precio.
Lee el total
Revisa el desglose por página, excluye las que no aplican y lee el precio final en la barra inferior.

Modelo de precios

Justo para ti
y para tu cliente

Cuatro tipos de página, cada uno con su modelo de cálculo óptimo.

📄 Texto B&N
Páginas con solo texto negro o gris, sin imágenes. Precio fijo configurable por página.
🎨 Texto a Color
Páginas con texto en color. Precio fijo independiente de cuánto texto haya.
🖼️ Imagen B&N
Imágenes en escala de grises. El precio escala con la cobertura real de tinta.
🎨 Imagen a Color
Precio mixto según la proporción real de píxeles cromáticos. Más justo que cobrar siempre el máximo color.
